# 契約管理アプリケーション デモ

AI契約管理システムのフロントエンドデモアプリケーションです。Next.js + TypeScript + Tailwind CSSで構築されています。

## 概要

このアプリケーションは、不動産賃貸契約の管理を効率化するためのAI支援システムのデモンストレーション版です。実際のバックエンド機能は実装されていませんが、UIとユーザー体験を確認できます。

## 主な機能

### 1. 契約書アップロード
- ドラッグ&ドロップでPDFファイルをアップロード
- OCR処理 → AI抽出の進行状況をリアルタイム表示
- 処理完了後、検証画面へ遷移

### 2. 抽出結果検証
- 左側にPDF原本、右側に抽出データを表示
- AI抽出データの信頼度を色分け表示（高：緑、中：黄、低：赤）
- PDF上のハイライト部分をクリックすると、対応するフォーム項目へジャンプ
- 全てのフィールドを編集可能

### 3. AI契約検索
- 自然言語で契約情報を検索
- タイピングアニメーションでAI回答を表示
- 関連度順に契約一覧を表示

### 4. AI契約分析
- 自然言語で契約データを分析
- 統計情報（平均賃料、件数、合計など）を表示
- 詳細データをテーブル形式で表示

### 5. レントロール作成
- 期間、ステータス、出力項目などを設定
- プレビュー表示
- Excel/CSV/PDF形式で出力（デモ）

### 6. 期限管理（カレンダー）
- 月次カレンダーで契約満了日を視覚化
- 今月のイベント一覧表示
- 期限別の契約件数サマリー

## 技術スタック

- **フレームワーク**: Next.js 15.5.4
- **言語**: TypeScript 5.9
- **スタイリング**: Tailwind CSS 3.4
- **UI**: React 19

## セットアップ

### 必要な環境
- Node.js 18以上
- npm または yarn

### インストール

```bash
# リポジトリをクローン
git clone <repository-url>
cd contract-manage-demo-next/app

# 依存パッケージをインストール
npm install

# 開発サーバーを起動
npm run dev
```

### アクセス

- ローカル: http://localhost:3000
- ネットワーク内の他のデバイス: http://<your-ip>:3000

## 主要コンポーネント

### UIコンポーネント
- `Button`: 4つのバリアント（primary, secondary, success, danger）
- `Card`: 白背景のコンテナカード
- `Badge`: 4つのバリアント（info, success, warning, danger）
- `ProgressBar`: 進行状況バー
- `Input`: フォーム入力

### レイアウトコンポーネント
- `Header`: ナビゲーションヘッダー
- `Sidebar`: サイドメニュー
- `DashboardLayout`: ダッシュボードレイアウト

## デモの特徴

### タイピングアニメーション
- 質問入力欄をクリックすると例文が自動タイピング
- AI回答が1文字ずつ表示される
- リアルな使用感を演出

### 進行状況の可視化
- ファイルアップロード処理の自動進行
- プログレスバーとステップ表示
- 処理完了後の自動遷移

### インタラクティブな検証画面
- PDF上のハイライトをクリックでフォームへジャンプ
- 信頼度による視覚的なフィードバック
- スムーズなスクロールアニメーション

## 注意事項

このアプリケーションはデモンストレーション用です：
- 実際のAI処理は行われません
- データベースへの保存は行われません
- ファイルのアップロードは実際には処理されません
- 全ての動作はフロントエンドのみで完結します
